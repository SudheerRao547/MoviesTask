Node.js Take-Home Assessment: Movie API with TypeScript


This project is a Node.js API implemented using TypeScript to fetch movies released in a specific year, sorted by descending popularity, with optional editor data. The solution integrates with the TMDB API and adheres to modern coding practices, including resilience, unit testing, and environment variable management.
